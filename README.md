# Registration System

A full-stack registration system demonstrating a modern multi-service architecture.

The system includes:

• Web client built with React  
• Mobile client built with React Native  
• Backend API built with Python (FastAPI)  
• AI service built with Node.js  
• Database using MongoDB  
• Deployment designed for Azure Cloud

The system allows users to register through the client application and receive a dynamic welcome message generated by an AI service.

---

# System Architecture

The system follows a microservice-style architecture where each service has a dedicated responsibility.

System Flow:

User  
↓  
React Web / React Native Mobile  
↓  
POST /auth/register  
↓  
Python API (FastAPI)  
↓  
Validate request  
↓  
Store user (MongoDB)  
↓  
GET /random-toast  
↓  
Node.js AI Service  
↓  
Generate welcome message  
↓  
Return response to Python API  
↓  
Frontend receives response  
↓  
Toast message displayed to the user

---

# API Endpoints

Python API (FastAPI)

GET /health  
Health check endpoint used to verify the API is running.

POST /auth/register  
Registers a new user.

Example request:

{
"name": "Ori",
"email": "ori@gmail.com",
"password": "123456"
}

Example response:

{
"message": "User registered successfully",
"user": {
"name": "Ori",
"email": "ori@gmail.com"
}
}

---

Node AI Service

GET /random-toast

Returns a random welcome message generated by the AI service.

Example response:

{
"message": "Welcome to the platform 🚀"
}

# Running the Project Locally

Start the AI Service (Node.js)

cd ai-node  
npm install  
node index.js  

Service runs on:

http://localhost:4001

---

Start the Python API

cd api-python  
python -m venv .venv  
.\.venv\Scripts\activate  
pip install fastapi uvicorn pydantic[email]  
uvicorn main:app --reload --port 4000  

API runs on:

http://localhost:4000

Swagger API documentation:

http://localhost:4000/docs

---

Start the Web Client

cd web  
npm install  
npm run dev  

Web application runs on:

http://localhost:5173
